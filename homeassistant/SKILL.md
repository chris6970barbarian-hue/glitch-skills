# Home Assistant

Control smart home devices via Home Assistant API.

## Setup

```bash
ha-cli setup <HA_URL> <TOKEN>
```

## Usage

```bash
ha-cli on <device>
ha-cli off <device>
ha-cli brightness 75 <device>
ha-cli scene movie
```

## Features

- Lights, switches, covers, climate control
- Scenes and scripts
- Natural language matching
